- Download node js and install on your sys

```
https://nodejs.org/en/
```

> Refer below command and uses:

| Command         |              uses              |
|-----------------|:------------------------------:|
| `npm install`   | To install all node dependency |
| `npm run start` |     To start local server      |
| `npm run webpack` |  To build for Production     |
